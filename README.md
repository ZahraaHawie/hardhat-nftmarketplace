# Hardhat NFT

## Overview (Backend)

Create a decentralized NFT marketplace:

1. `listItem`: List NFTs on the marketplace
2. `buyItems`: Buy the NFTs on the Marketplace
3. `cancelItem`: Cancel a listing
4. `updateListing`: Update Price
5. `withdrawProceeds`: Withdraw payment for my bought NFTs

### Steps

1. NftMarketplace.sol
2. 01-deploy-nft-marketplace
3. BsaicNft.sol (We need some NFTs)
4. 02-deploy-basic-nft.js
5. tests for NFTmarketplace.sol
6. scripts > mint and list
